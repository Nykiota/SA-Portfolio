# SA-Portfolio
Этот репозиторий содержит проекты по System Analyst, в которых представлены весь доступный мне инструментарий

| [2. Моделирование данных и работа с базой данных для он-лайн магазина](practicum_projects/on-line_store) | В рамках проекта было необходимо составить логическую модель и словарь данных на основании спецификации требований. Сделать нормализацию | *- ERD (нормализация до 3НФ); Use Case* |
| [4. Упорядочить заказ билетов для он-лайн агрегатора](practicum_projects/flight_ticket_aggregator) | В рамках проекта было разобраться в последовательности операций для последующей интеграции| *- Sequence диаграмма* |

| [6. | [7. Подготовка API для стриминогового сервиса](practicum_projects/streaming_service) | В рамках проекта было необходимо провести интервью со стейкхолдерами, смоделировать данные, протестировать API и передать в разработку | *- DFD (контекстная и логическая); - тикет на разработку* |
| [8. Разработка приложения Умный Дом](practicum_projects/smart_home) | В рамках проекта было необходимо разработать требования к приложению и создать прототип пользовательского интерфейса приложения | *- Карта пользовательских историй;- ERD; - DFD;- словарь данных;- диаграмма структуры интерфейса; - ПМИ;- Прототипы интерфейсов* |
