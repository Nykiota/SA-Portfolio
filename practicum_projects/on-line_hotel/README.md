## Cайт для отеля 

### Контекст 

Наша команда готовится к запуску веб-версии сервиса бронирования капсульных отелей. В этих отелях клиент может остановиться на несколько часов, чтобы вздремнуть и набраться сил.
Сайт для сервиса  практически готов и уже перешёл на этап тестирования. Сейчас необходимот проверить несколько функций приложения: регистрацию на сайте, личный кабинет пользователя, бронирование отеля и управление бронированием. 
Вам предстоит протестировать функциональность сайта отеля, чтобы проверить её готовность для передачи пользователям. 
В рамках проведенного тестирования, был выявлен ряд ошибок на сайте бронирования капсульных отелей, которые были исправлены разработчиками. Поэтому нам нужно сейчас начать подготовку программы и методики испытаний (ПМИ), а также руководства пользователя, чтобы заказчик смог проверить соответствие выполненных работ предъявленным требованиям. 

### Выполненные задачи

1) Тест-кейсы
2) ПМИ сервиса.
3) Руководство пользователя сервиса.
