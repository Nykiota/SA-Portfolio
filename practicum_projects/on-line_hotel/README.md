## Cайт для отеля 

### Контекст 

Команда RANA.Software готовится к запуску веб-версии сервиса бронирования капсульных отелей «Капсула». В этих отелях клиент может остановиться на несколько часов, чтобы вздремнуть и набраться сил.
Сайт для сервиса «Капсула» практически готов и уже перешёл на этап тестирования. В этой версии вам предстоит проверить несколько функций приложения: регистрацию на сайте, личный кабинет пользователя, бронирование отеля и управление бронированием. 
Команда на данный момент тестирует новую доработанную версию сайта 2.2. 
Вам предстоит протестировать функциональность сайта отеля, чтобы проверить её готовность для передачи пользователям. 
После обнаружения ошибок на сайте бронирования капсульных отелей, которые мы нашли после проведения теста, нам нужно уже сейчас начать подготовку программы и методики испытаний (ПМИ), а также руководства пользователя, чтобы заказчик смог проверить соответствие выполненных работ предъявленным требованиям. 

### Выполненные задачи

1) ПМИ сервиса.
2) Руководство пользователя сервиса.
