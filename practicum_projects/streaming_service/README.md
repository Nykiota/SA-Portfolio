## Стриминговый сервис

### Контекст
С того момента, когда вы изучали API Otium, кое-что изменилось. Теперь для обмена данными используется REST API, а документация API хранится в Swagger. Вам нужно помочь Дане и Айгуль заново изучить архитектуру Оtium и существующие API и поставить задачи на доработку API.
Для этого требуется:
- Изучить интервью с представителем Otium. Вам нужно понять требования заказчика. После интервью проверите, насколько это получилось.
- Создать диаграмму последовательности. Она поможет разработчикам реализовать новую функцию сервиса.
- Протестировать существующее API в Swagger. После тестирования нужно будет ответить на несколько вопросов.
- Поставить задачу на разработку новой операции API.

### Выполненные задачи
1) DFD
2) Проверка XML-схемы
3) Тикет на разработку
